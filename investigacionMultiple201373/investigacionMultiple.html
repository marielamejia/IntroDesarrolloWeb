Como parte del HTML existen distintos tipos de categorías que se le pueden dar a los elementos o etiquetas. Estos se conocen como elementos de línea y de bloque.

Investigar qué son los elementos de línea y elementos de bloque. Poner por lo menos 5 ejemplos de cada uno.

Elementos de línea

Los elementos de línea se usan generalmente para dar formato o significado dentro de un texto. Esto se debe a que utilizan solo el espacio necesario para su contenido, sin la necesidad de forzar un salto de línea. Algunos ejemplos de elementos de línea son:

  • <a>
  • <b>
  • <em>
  • <img>
  • <label>

Elementos de bloque

Los elementos de bloque son aquello que, aunque su contenido no lo requiera, utilizan todo el ancho de la página. Dentro de un elemento de bloque se pueden anidar tanto elementos de bloque como elementos en línea. Algunos ejemplos de elementos de bloque son:

  • <div>
  • <section>
  • <h1>
  • <p>
  • <ul>

Inputs y label

La etiqueta <label > sirve para asociar un texto descriptivo a un campo de formulario. Además, mejora la accesibilidad porque al hacer clic en el label, se activa el campo correspondiente.

Por otro lado, la etiqueta <input > se utiliza para que los usuarios puedan ingresar datos en un sitio web. Para esta etiqueta, usamos el atributo type, el cual indica el tipo de entrada que se espera del usuario, como texto o números.

Además de type, algunas otras propiedades que tienen los inputs son:

  • name: el nombre del campo en el formulario
  • value: indica si hay algun valor por defecto
  • placeholder: es un texto de ayuda en el campo
  • required: indica si el campo es obligatorio
